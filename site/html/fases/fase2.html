CSS

CSS (sigla para Cascading Style Sheets, ou Folhas de Estilo em Cascata em português) é uma linguagem usada para estilizar páginas da web. Ele é utilizado junto com HTML para controlar a aparência visual de elementos como textos, cores, margens, espaçamentos, tamanhos, layouts e muito mais.

Para quê ele serve?

Ele separa o conteúdo (feito em HTML) da apresentação visual, permitindo que os desenvolvedores: Apliquem cores e fontes específicas. Definam layouts com colunas, grades e posicionamentos.

Criem efeitos visuais como animações e transições.

Adaptem o site para diferentes dispositivos (como celulares, tablets e desktops).
