JavaScript

JavaScript é uma linguagem de programação usada principalmente para tornar páginas da web interativas. Ele é executado no navegador (como Chrome, Firefox, Safari etc.) e permite que você crie funcionalidades como:



Animações em páginas;

Validação de formulários (como checar se o e-mail foi preenchido corretamente);

Botões que reagem a cliques;

Atualizações dinâmicas de conteúdo sem recarregar a página (como em chats, redes sociais, etc.); Jogos, gráficos, e muito mais.

HTML: estrutura da página (como ossos);
CSS: aparência da página (como roupas);
JavaScript: comportamento da página (como o cérebro).

Importante:

Não confunda JavaScript com Java — apesar dos nomes parecidos, são linguagens diferentes com propósitos diferentes.
