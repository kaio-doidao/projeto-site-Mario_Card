resumo

HTML (HyperText Markup Language): é a linguagem que estrutura o conteúdo da página. Com ele, você cria títulos, textos, imagens, links e tudo o que aparece no navegador. Pense no HTML como os ossos da página.



CSS (Cascading Style Sheets): é a linguagem que define a aparência da página. Com CSS você controla cores, fontes, tamanhos, espaçamentos, layouts e efeitos visuais. Ele é como as roupas e o estilo da página.



JavaScript: é a linguagem que torna a página interativa. Permite animações, validação de formulários, atualizações dinâmicas e respostas a ações do usuário. JavaScript é o cérebro que faz tudo reagir.
