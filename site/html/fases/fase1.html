HTML

1. O que exatamente é o HTML? HTML é uma linguagem de marcação usada para estruturar conteúdo na web. A palavra “markup” significa “marcar” ou “etiquetar” o texto para que o navegador saiba como exibir cada parte.

1.2. Elementos e Tags Um documento HTML é formado por elementos representados por tags. Uma tag de abertura marca o início de um elemento, Uma tag de fechamento marca o fim do elemento, Entre elas, colocamos o conteúdo, que será exibido na página.

3. Escreva a estrutura básica do HTML:

DOCTYPE html: Declara que o documento é HTML5, a versão mais recente.

html lang="pt-BR": Marca o início do documento e define o idioma.

head: Contém metadados, título, links para arquivos CSS, scripts, etc.

meta charset="UTF-8": Define a codificação de caracteres para que os acentos e símbolos funcionem bem.

title: Título da página que aparece na aba do navegador.

body: Onde você coloca todo o conteúdo visível da página.
